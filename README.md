# 中文 90일 - TSC 5-6급 학습 앱

## 배포 방법

1. 이 저장소를 GitHub에 업로드
2. Settings → Pages → Source: main branch / root
3. Save 후 1-2분 대기
4. `https://[username].github.io/[repo-name]` 으로 접속

## 기능
- 90일 커리큘럼 (Phase 1 워밍업 / Phase 2 심화 / Phase 3 실전)
- 어휘 학습 + 성조 색깔 표시
- 플래시카드 (앞뒤 뒤집기)
- 발음 듣기 (팝업 → 파파고/구글번역)
- 단어 퀴즈
- 발음 녹음 & 비교
- 학습 통계 & 그래프
- 북마크
- 진도 자동 저장 (localStorage)
